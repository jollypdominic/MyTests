# 📅 Team Attendance Calendar 2026

A comprehensive web-based attendance tracking system for managing daily attendance of 4 team leads throughout 2026.

## ✨ Features

### Core Functionality
- **Interactive Calendar View** - Visual monthly calendar with easy navigation
- **4 Lead Tracking** - Simultaneous attendance tracking for 4 team leads
- **Multiple Status Types** - Present, Sick Leave, Casual Leave, Vacation, Absent
- **Visual Indicators** - Color-coded dots for quick status identification
- **Customizable Names** - Personalize lead names for your team

### Reporting & Analytics
- **Monthly Reports** - Comprehensive statistics for each month
- **All Present Days** - Track days when all 4 leads were present
- **Leave Tracking** - Monitor sick leaves, casual leaves, and vacation days
- **Individual Statistics** - Per-lead breakdown of attendance
- **CSV Export** - Export monthly data for external analysis

### Data Management
- **Local Storage** - All data saved in browser (no server required)
- **Persistent Data** - Data remains even after closing browser
- **Easy Backup** - Export data anytime as CSV
- **Clear Data Option** - Reset all data when needed

## 📖 How to Use

### 1. Set Up Lead Names

1. Look at the left panel "Team Leads"
2. Enter the names of your 4 leads in the input fields
3. Names are automatically saved

### 2. Mark Daily Attendance

1. Click on any date in the calendar
2. A modal will open showing all 4 leads
3. Select status for each lead:
   - **Absent** - Not present (default)
   - **Present** - Working normally
   - **Sick Leave** - On sick leave
   - **Casual Leave** - Taking casual leave
   - **Vacation** - On vacation
4. Click "Save Attendance"

### 3. View Attendance Status

- Each calendar day shows 4 colored dots (one per lead)
- **Gray** = Absent
- **Green** = Present
- **Yellow** = Sick Leave
- **Blue** = Casual Leave
- **Red** = Vacation

### 4. Navigate Months

- Use **← Previous** and **Next →** buttons
- View any month in 2026
- Today's date is highlighted in yellow

### 5. Generate Monthly Report

1. Navigate to the desired month
2. Click **📊 Monthly Report** button
3. View comprehensive statistics:
   - Days all 4 leads were present
   - Total sick leaves taken
   - Total casual leaves taken
   - Total vacation days
   - Individual lead statistics

### 6. Export Data

1. Navigate to the desired month
2. Click **💾 Export Data** button
3. CSV file downloads automatically
4. Open in Excel, Google Sheets, or any spreadsheet software

### 7. Clear All Data

1. Click **🗑️ Clear All** button
2. Confirm the action
3. All attendance data will be permanently deleted

## 🔧 Troubleshooting

### Data Not Saving?
- Check if browser allows Local Storage
- Try a different browser
- Clear browser cache and try again

### Calendar Not Displaying?
- Ensure JavaScript is enabled
- Try refreshing the page (F5)
- Check browser console for errors

### Export Not Working?
- Check browser's download settings
- Allow pop-ups for this page
- Try a different browser

### Lost Data?
- Data is stored per browser
- Using a different browser = different data
- Always export important data as backup
